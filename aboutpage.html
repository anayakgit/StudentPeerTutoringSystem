Student Peer Tutoring System

A web-based application designed to facilitate peer tutoring among students in various courses. The system integrates user-friendly interfaces and efficient database management to enhance learning experiences.

Key Features

  • Course selection and matching
  • User-friendly interface
  • Real-time chat support
  • Comprehensive user profiles
  • Secure user authentication

Database Tables

The project utilizes the following tables:

  • student: Stores information about registered students, including name, year, branch, email, and login credentials.
  • tutor: Contains details about tutors available for peer tutoring, including expertise in various courses.
  • class: Manages information related to specific classes or courses offered within the system.
  • courses: Stores data about available courses, including course names and descriptions.
